What is Python?

What are Python and Pip?

  • Python is the programming language we will use for almost everything in this course. Think of it as the engine for our car. It does all the actual work, like processing data, running calculations, and creating visualizations.

  • Pip is Python‚Äôs Package Installer. Think of it as the mechanic who installs new parts in our car‚Äôs engine. When we need our car to do something new (like work with a specific type of data), we‚Äôll use pip to install a ‚Äúpackage‚Äù that adds that new ability.

How to Open Your Terminal

The terminal (also called the command line or shell) is a text-based way to talk directly to your computer. Instead of clicking on icons, you type commands. It‚Äôs fast, powerful, and a standard tool for all developers. Click on the tab below that matches your operating system for instructions on how to open it.

  • Windows (WSL2)
  • Windows (Standard)
  • macOS
  • Linux

This is your primary environment for this course.

  1. Press the Windows key.
  2. Type Ubuntu (or the name of your installed Linux distribution).
  3. Select the app to open your Linux terminal inside Windows.

Use this if you are not using WSL2. PowerShell is recommended over the classic Command Prompt.

  1. Press the Windows key.
  2. Type PowerShell.
  3. Select Windows PowerShell.
Compatibility

While many commands are the same, some instructions in this course may need minor changes if you are using PowerShell instead of the Linux-based terminal in WSL2.

The terminal app on macOS is simply called Terminal.

  1. Press Cmd + Space to open Spotlight search.
  2. Type Terminal and press Enter.
  3. You can also find it in Applications > Utilities > Terminal.

Most Linux distributions make the terminal easy to find.

  1. Try the common keyboard shortcut: Ctrl + Alt + T.
  2. If that doesn‚Äôt work, open your applications menu and search for ‚ÄúTerminal‚Äù, ‚ÄúConsole‚Äù, or ‚Äúxterm‚Äù.

Step 1: Check if Python is Already Installed

Many computers come with Python pre-installed. Let‚Äôs check if you already have it. Type this command into your terminal and press Enter:

python3 --version

If you see a version number like Python 3.8.10 or higher, you are in great shape! You can skip to Step 3. If you see an error or a version older than 3.8, continue to the next step.

Step 2: How to Install Python

Click on the tab below that matches your computer‚Äôs operating system.

  • Windows (WSL2 - The Best Way)
  • macOS
  • Linux (Ubuntu/Debian/Bestway)
  • Windows (Direct - Not Recommended)

This is the most reliable way to use Python on Windows for data science. It keeps your Python environment clean and separate from Windows.

  1. Requirement: This method uses the Windows Subsystem for Linux (WSL2). If you haven‚Äôt installed it yet, please follow the Official Microsoft WSL Installation Guide and choose Ubuntu.

  2. Update Your Software List: In your Ubuntu terminal, run this command to get the latest list of available software.

    sudo apt update

  3. Install Python and Friends: This command installs Python (python3), its package installer (python3-pip), and a tool for keeping projects separate (python3-venv). The -y at the end just says ‚Äúyes‚Äù to any questions it might ask.

    sudo apt install python3 python3-pip python3-venv -y

The best way to manage software like Python on a Mac is with a tool called Homebrew.

  1. Install Homebrew: If you don‚Äôt have it, open your Terminal app and paste the single command from the official Homebrew website. It will guide you through the installation.

  2. Install Python: Once Homebrew is installed, run this simple command:

    brew install python

    Homebrew handles all the tricky details for you.

Most Linux systems have Python, but this ensures you have all the necessary helper packages.

  1. Update Your Software List:

    sudo apt update

  2. Install Python and Friends:

    sudo apt install python3 python3-pip python3-venv -y
Proceed with Caution

This method can be tricky. A system setting called PATH tells your computer where to find commands. If this isn‚Äôt set right during installation, your computer won‚Äôt be able to find python when you type it in the terminal. We strongly recommend the WSL2 method instead.

  1. Go to the official Python website‚Äôs download page for Windows.
  2. Download the latest stable installer (e.g., ‚ÄúWindows installer (64-bit)‚Äù).
  3. Run the installer. This is the most important step: On the very first screen of the installer, you MUST check the box at the bottom that says ‚ÄúAdd python.exe to PATH‚Äù. If you miss this, you will have problems later.

Step 3: Verify Your Installation

Let‚Äôs do a final check to make sure everything is working. Close your terminal, open a new one, and run these two commands one by one:

python3 -m pip --version

You should see version numbers for both python3 and pip. If you do, congratulations! Your engine is installed and ready to go.
